### 1. 인프런(공공데이터분석하기)
#### 정규 표현식으로 데이터 전처리 하기
- 정규 표현식 라이브러리
  - 파이썬 내장 라이브러리이다.
  
  ```python
  import re

  def get_string(text):
    if pd.isna(text):
        return text
    
    text = re.sub("[\d\.'\-(nan)\[\]\/]","",text)
    text = re.sub(" ","",text)
    text = re.sub(",{2,}","",text)
    text = re.sub("\+",",",text)
    return text
  ```
