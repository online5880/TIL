### 2. Django
#### Template Tags
- DTL(Django Template Library)
  - Django Template Language(DTL)는 Django에서 템플릿을 작성할 때 사용하는 언어
  - HTML에 데이터를 삽입하여 동적 웹 페이지를 만들 수 있도록 도와준다. 
- DTL의 기본 구조
  - DTL은 템플릿 파일에서 변수 출력, 조건문, 반복문 등을 지원함. 
  - 일반적인 HTML 파일에 `{{ }}`와 `{% %}` 문법을 통해 파이썬 코드를 삽입하는 방식
